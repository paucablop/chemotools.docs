Cite chemotools
===============

.. toctree::
   :maxdepth: 1
   :hidden:

Chemotools has been peer-reviewed and published in the Journal of Open Source Software (JOSS). The paper is available at:
https://doi.org/10.21105/joss.06802

.. note::

   If you use chemotools in your research or software and would like to cite it, please use the following BibTeX entry:

   .. code-block:: bibtex

      @article{Lopez2024,
      author = {Lopez, P. C.},
      title = {chemotools: A Python Package that Integrates Chemometrics and scikit-learn},
      journal = {Journal of Open Source Software},
      volume = {9},
      number = {100},
      pages = {6802},
      year = {2024},
      doi = {10.21105/joss.06802},
      url = {https://doi.org/10.21105/joss.06802}
      }
